モジュールやテーマのアップデートを短時間で行えるモジュールです。

○ 動作環境

- PHP 5.2.0以上 (必須)
- safeモードで無いこと (ほぼ必須：GitHubからのダウンロード)
- cURLエクステンション (必須)
- zip 拡張関数：--enable-zipでphpがコンパイルされている (推奨)
- MySQL 5.0 以上 (必須)
- Xoops Cube legacy 2.2.0 以上
