D3 Forum

D3 Forum is a duplicable module used to create and manage forums, or message boards discussions. This module provides features comparable to popular community forum software. It also provides the comments-integration functionality for D3 modules e.g. Pico.

Creating discussion forum categories helps managers and users to quickly access the required information. Generally, the structure of a forum covers particular categories. See more about this in documentation section structure.

Administrators can assign user groups and individual permissions into various categories. Users with appropriate access rights can discuss one specific topic with other members of the platform.

New Category and Forum

D3 modules are designed to create categories and contents on the front end.

Click "New Category" and fill in the details in the form page.

  1. Category title
  2. Category description (allows format text with BBCode)
  3. Parent category (default top-level category)
  4. Category order (weight position, default sorting by category ID)
  5. Category options, (more about this below in section options).

When you‚Äôve included all these details, you can simply click "Submit" at the bottom of the form, and your Category will be created.

Category, subcategories and forums inherit permissions from the nearest parent category, in the absence of individual permissions, from the Top-level category, by default.

Moderation

Only administrators can create categories and forums. However, you can assign users and roles to moderate and create forums (either individually, or for all categories on your module instance). For example, you may grant users a specific role "moderator" with the ability to create forums.

  1. Click the button, or icon "Edit category"
  2. Click "Category permissions" to manage users permissions

Overridable options

Categories, subcategories and forums inherit the options from the Top-level category, by default. However, you can customize the default options when creating or editing a category or forum.

Override preferences by writing a new line for each option.
(option name):(option value)

Example to show breadcrumbs (1=on, 0=off):
show_breadcrumbs:1

Preferences

Module Preferences focus primarily on how the forums, forum topics, and comments are set on the site.

The D3 Forum module comes with several options set for a new forum.

Review the list of options and define your preferences.

If you are familiar with themes and templates, you can duplicate the template files of the module to be customized.

You can install multiple instances of D3Forum using the X-Update Manager.
