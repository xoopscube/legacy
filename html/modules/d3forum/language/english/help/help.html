D3 Forum

D3 Forum is a duplicable module used to create and manage forums, discussions and message boards with features comparable to the most popular community forums software. This module also provides the functionality for comments-integration of D3 modules e.g. Pico.

Usage

First, create a Category and a forum in the front-end.

Then you get access to the module advanced options and permissions.

Forum Blocks

The module allows you to add blocks with posts, topics and forums in the different modules and pages of your site.

Each block has its own options and template, permissions and cache settings.

Related content

Preferences

Module Preferences focus primarily on how the forums, forum topics, and comments are set on the site.

The D3 Forum module comes with several options set for a new forum.

Review the list of options and define your preferences.

If you are familiar with themes and templates, you can duplicate the template files of the module to be customized.

You can install multiple instances of D3Forum using the X-Update Manager.
