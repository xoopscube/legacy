XOOPS Cube Legacy 2.2 KICK START GUIDE

Overview

A kick start guide to XOOPS Cube Legacy if you are running this cpanel for the first time.
The goal of this kick-start guide is to help new site administrators get orientated to what this position does and what responsibilities it entails. It provides the minimum amount of information needed, get started and begin working with the XOOPS Cube Legacy 2.2 system. Other detailed instructional materials are available and will be highlighted throughout.

Read this guide!


Outline

  • This XOOPS Cube Legacy modules classes ensure compatibility with different parts of XOOPS 2.0.x.
  • And provide various features alike Modules installation, related menus, blocks management and a theme engine for administration.
