Profile Module

The module Profile features a simple user-friendly interface to manage custom fields for the User profile. You can create new user profile fields : checkbox, menu of choices, text area, text input and make any of these fields required.

New profile fields will appear on each user's profile page only to user groups selected in "Access groups" otherwise the fields are not visible, in which case only the administrator can see the field. To create a new profile field, select the profile field Type you require from the dropdown list.

You can set the order in which your custom profile fields appear the User profile page.

Custom fields

  • String - allows a series of characters to be stored.
  • Text - textarea with editor.
  • Int (Integers) - allows value containing only decimal digits.
  • Date - allows a specific date format.
  • Checkbox - allows a selection 0 or 1, true or false.
  • Select box - allows multiple options, see the example.

Add Profile Field

Click the button "New Field" and fill the form.

  • Field form name. This is used only internally!
  • Label
  • Type. This can not be edited later.
  • Validation [ email ]
  • Required?
  • Display . Check this option to display the field on User profile
  • Order
  • Description
  • Access groups
  • Options. Depends on the Type of fiel.Exemple checkbox :
    Define the display string when values are "checked" and "unchecked". Separate values delimited by the pipe character ("|"). When empty, "Yes" and "No" is used.

View Profile Field

Click the button "View" to display properties of custom field.

Edit Profile Field

Click the button "Edit" to change the properties. However, you can not edit the Type of custom field.

Delete Profile Field

Click the button "Delete" to remove the field from the module and user profile.
