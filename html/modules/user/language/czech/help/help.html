User module

Overview

This module offers various feature about users to Legacy System base module. Offering feature are Xoops Authentication, User Registration, User Profile Edit, User Profile Display, Blocks about users, User Management and others.

Install
You don't need to do something to install in most cases, because this package might have been already installed correctly. If you need more informations, go to "Installation Guide".
