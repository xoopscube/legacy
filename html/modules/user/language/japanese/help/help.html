ユーザーモジュール

概要

このモジュールはユーザに関する様々な機能を Legacy BASE （互換モジュール）に提供します。提供する機能は XOOPS 認証、ユーザー登録、ユーザーアカウントの編集、ユーザーアカウントの表示、ユーザー周りのブロックと、管理者のためのユーザー管理などです。

インストール

このモジュールは既に正しくインストールされているはずですので、たいていの場合、あなたはインストールのために何かをする必要はありません。さらに詳しい情報については、「インストールガイド」をご覧ください。
