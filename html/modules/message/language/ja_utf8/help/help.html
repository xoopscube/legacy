Overview

This module Message provides Private Messages functions with extra features and enhancements. Private Message is a type of communication wherein the message can only be viewed or read by a specific recipient. This is also known as Direct Message (DM), or Personal Message.

Features

  • Add Block to display messages of Inbox and Outbox.
  • Messages status, read, unread, lock.
  • Search messages by status or username from Inbox and Outbox.
  • Option to purge messages automatically.
  • Settings to define pagination (messages per page).
  • Forward message to email account.
  • Blacklist management.

Blocks

Install the block 'New Messages' for registered users.

Preferences

Administrators can set the default number of messages per page.

Delay to purge messages and set if unread messages are also deleted.

Replace the default user profile template with this module's template
when installing the "myfriend" module and, or the "social network" module.
