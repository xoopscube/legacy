Banner Management

Allows you to manage adverts and displays banners in your website.

You can place a banner of any type, on any location of the page, of any module.

You may show a banner to visitors a certain number of impressions.

An impression is a metric used to quantify the number of digital views of a client banner.

Banner Smarty Tags

Smarty generates web content through the placement of special Smarty tags within a document.

The Banner tags can be added to Themes, Templates or Blocks.

Smarty tags will be processed and substituted with content.

The .htaccess File

.htaccess files are plain-text configuration files used by the Apache HTTP web server. They allow users to set directory level options without requiring access to the httpd.conf file, including www rewrite, hotlink prevention, custom error pages, caching, and redirects.
As such it is required that your server uses Apache, and a web host that allows htaccess files (the most popular hosts do).

Template Management

The distribution package comes with a Default Template Set.

When modules are installed, their templates become part of this Default Template set.

The list of Template Sets is displayed in Template Management.

The Default Template Set cannot be deleted nor modified !

Custom Template Set

Clone the Default Template Set to create a new Template Set

Edit the templates in your new Template Set

Once you have a new Template Set customized, you can activate it with the "Select" button.

Template File Management

The list of template files allow download and editing of each file.

You can upload Template Set archives in tar format through the admin interface.

Theme Selection

Theme Management allows you to "Select" the theme that you want to use as the default theme or main theme of your site.

Any theme that is "checked" in the "Control" column, will be available to users in the block "Themes".

If you install the block "Themes" and grant users permission, they can then select their favorite theme and customize the look and feel of the website.

Install a Theme

Distributed themes are mostly compressed. To use a theme, the uncompressed files must be uploaded into the directory public_html/themes/.

The Theme folder must include theme.html and manifesto.ini.php

Additional Themes can be installed from X-Update Store, from theme-developers' websites or your custom design.

Uninstall a Theme

The render system manages themes without using the database.

So, to delete a theme, simply delete its directory from the "themes" directory.

AMP HTML Specification

AMP HTML is a subset of HTML for authoring content pages such as news articles in a way that guarantees certain baseline performance characteristics. Being a subset of HTML, it puts some restrictions on the full set of tags and functionality available through HTML, but it does not require the development of new rendering engines: existing user agents can render AMP HTML just like all other HTML.

Also, AMP HTML documents can be uploaded to a web server and served just like any other HTML document; no special configuration for the server is necessary. However, they are also designed to be optionally served through specialized AMP serving systems that proxy AMP documents.

BBCode

BBCode is a simple way to format your entries (like bold text, italic text, links) in forums. BBCode was implemented as method of providing a safer and easier way of allowing posts to be formatted on forums.

Formatting Posts

BBCode is short for Bulletin Board Code. It is used as a way for formatting posts made on message boards, blogs and more. It is similar to HTML in the sense that in BBCode one does also use tags to format something specific (contained within the tag). In BBCode tags are indicated by rectangular brackets surrounding a keyword, which is in turn transformed into HTML before being delivered to a web browser. Since its creation, BBCode has been used in many different portals. Any developer can implement it for free and anyone can use it.

Markdown

Markdown is a tool that converts simple text characters into HTML. It was created to facilitate writing on the web. It is very similar in purpose to BBCode, being created later (in 2004). Unlike BBCode, Markdown aims to avoid the use of tags wherever possible, as if the writing was just plain text.

Markdown vs BBCode

In general, Markdown is not necessarily a competitor to BBCode, just a different and complementary way of writing on the web, which may be more suitable in certain contexts.

In general, the writing difficulty can be ordered like this:

Text < Markdown < BBCode < HTML

The further to the right, the more features, and the further to the left, the simpler.

Markdown usage requires the installation of a plugin, preload or module.
