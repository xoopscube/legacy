Iconography

X-icons.css (inline SVG>) and jQuery UI icons can be used to illustrate action, status and identity.

Options

  1. Inline SVG icons
  2. Customized with CSS
  3. Color, size, spacing and position
  4. jQuery UI icons

Preview

jQuery UI provides a number of icons that can be used by applying class names to elements. The class names generally follow a syntax of .ui-icon-{icon type}-{icon sub description}-{direction}. For example, the following will display an icon of a thick arrow pointing north:

ui-icon-folder-open

Inline SVG icons allows one single version of an icon to be customized dynamically using CSS or JavaScript. They can also be sized in relation to the typography guidelines.
