Module MyCKEditor

Sommaire

Ce module est un exemple de mise en œuvre d'un éditeur WYSIWYG comme module XOOPS Cube Legacy.

WYSIWYG (What You See Is What You Get) implique une interface utilisateur qui permet lors de la création de visualiser le contenu de manière très similaire au résultat final.

Gérez votre contenu Web facilement avec le nouveau éditeur CKEditor pour les textarea de XOOPS Cube Legacy.

CKEditor c'est quoi au juste?

CKEditor est un éditeur de texte pour être utilisé à l'intérieur des pages web. Il s'agit d'un éditeur WYSIWYG, ce qui signifie que le texte en cours d'édition sur la page ressemble autant que possible su résultat final que verront les utilisateurs. Il apporte les fonctionnalités d'édition web trouvés sur les applications d'édition comme Microsoft Word et OpenOffice.

CKEditor vs Word

CKEditor n'est pas une application de bureau comme Microsoft Word ou OpenOffice. C'est un composant pour être utilisé par les développeurs pour améliorer leurs applications. C'est un éditeur pour être utilisé à l'intérieur des pages web.

Comment utiliser

Requis

XOOPS Cube Legacy 2.2 ou version ultérieure.

Mettre en œuvre CKEditor sur les textarea en éditant les Templates.

Caractéristiques Principales

La barre d'outils de CKEditor peut être entièrement personnalisé.
démo

Installation
  1. Clone / Dupliquer vos Templates pour créer un nouvel ensemble.
  2. Modifier les Templates avec des textarea de tous les modules installés en ajoutant un nouveau paramètre editor=html

<{xoops_dhtmltareaeditor=html}>

Note :N'oubliez pas d'activer votre nouveau ensemble de Templates.
Le module MyCKEditor ne fera aucune conversion des données existantes.
Les données seront automatiquement affichées dans leur format original.
